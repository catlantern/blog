## ⚠️ 今日重点关注

### 1. 思科 Catalyst SD-WAN Manager 高危漏洞 CVE-2026-20245 正被积极利用
- **攻击复杂度**：中。漏洞利用需要本地访问及 netadmin 权限，但攻击者可通过组合旧版身份验证绕过漏洞或窃取凭据达成条件。
- **影响范围估算**：全球范围内使用思科 Catalyst SD-WAN Manager 的企业和组织，包括政府、金融、能源等关键行业。
- **建议优先级**：紧急。漏洞尚无补丁，思科仅建议升级至最新可用版本并检查设备配置篡改。建议用户立即采取缓解措施。
- **相关联历史事件**：旧版身份验证绕过漏洞曾由网络间谍组织 UAT-8616 利用；Mandiant 报告该漏洞。

### 2. 微软 GitHub 仓库供应链攻击，至少 70 个开源项目被植入窃密木马
- **攻击复杂度**：中等。攻击者成功入侵微软官方仓库并通过社会工程或凭据窃取获得提交权限。
- **影响范围估算**：使用受影响 AI 开发工具（Claude Code、Gemini CLI、VS Code 等）的开发者群体，大量云服务和用户数据面临风险。
- **建议优先级**：紧急。建议开发者立即检查是否使用了受影响仓库，更改相关密码和凭证。
- **相关联历史事件**：近几个月来针对热门开源项目的供应链攻击频发，这是影响面最广的一次。

### 3. 日本连发两起大规模个人信息泄露事件（1090 万 + 51 万人）
- **攻击复杂度**：低。两起事件均由物理介质处置不当导致，非技术性攻击。
- **影响范围估算**：九州电力事件涉及 1090 万用户，北海道医疗事件涉及最多 51 万人，合计超 1100 万人。

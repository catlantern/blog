## 🔥 高危漏洞预警

> **标题**：从路由器到服务器：潜伏 13 年的 U-Boot 漏洞曝光，威胁数百万设备
> **来源**：IT之家（转引 Binarly 安全报告）
> **发布时间**：2026-07-11
> **摘要**：固件安全公司 Binarly 披露了开源引导程序 U-Boot 中存在的六个高危漏洞，编号为 BRLY-2026-037 至 BRLY-2026-042。其中两个可导致任意代码执行，四个可引发拒绝服务（DoS）攻击。易受攻击的代码自 2013 年 7 月发布的 U-Boot v2013.07 版本起便已存在，影响超过 50 个稳定版本及无数下游厂商分支固件。漏洞出现在 FIT 镜像签名验证核心代码中，攻击者可在签名验证完成前通过构造恶意镜像触发漏洞。Binarly 研究团队已在 QEMU ARM 仿真环境中成功验证了代码执行的利用可行性。
> **链接**：https://www.ithome.com/0/975/531.htm
> **标签**：`#U-Boot` `#Binarly` `#高危漏洞` `#BRLY-2026` `#供应链安全`

> **标题**：Claude Code 被国家信息安全漏洞库（NVDB）认定存在安全后门隐患，危害严重
> **来源**：凤凰网 / 网易（新智元报道）
> **发布时间**：2026-07-11（NVDB 于 7 月 8 日发布预警，持续发酵中）
> **摘要**：国家信息安全漏洞库（NVDB）发布官方风险提示，认定 Claude Code 存在“安全后门隐患，危害严重”。开发者逆向工程发现 Claude Code 2.1.91 至 2.1.196 版本内置间谍软件，未经用户同意向远程服务器回传地域、身份标识等敏感信息。Anthropic 团队成员承认这是 2026 年 3 月启动的“实验性”反滥用措施，该行为已在 7 月 2 日的新版本中被移除。阿里巴巴已于 7 月 10 日起全面停用 Claude 系列产品。

> **标签**：`#Claude Code` `#NVDB` `#Anthropic` `#后门` `#间谍软件` `#AI安全`

## ⚠️ 今日重点关注

**1. Claude Code 间谍软件后门被 NVDB 认定危害严重**
- **攻击复杂度**：低。后门以“反滥用措施”名义内置于正常版本中，用户正常下载使用即触发数据回传，无需额外攻击链。
- **影响范围估算**：全球范围内使用 Claude Code 2.1.91 至 2.1.196 版本的开发者。阿里巴巴等企业已全面停用受影响产品，国内受影响用户数量较大。
- **建议优先级**：紧急。建议所有使用 Claude Code 的单位和用户立即排查版本并升级至 2.1.196 以上版本，同时排查是否存在敏感信息已被回传。
- **相关联历史事件**：此前 SolarWinds 供应链攻击事件中同样出现开发工具被植入后门的情况，波及范围广泛。

**2. U-Boot 六个高危漏洞曝光，潜伏 13 年影响数百万设备**
- **攻击复杂度**：中。漏洞存在于 FIT 镜像签名验证核心代码中，攻击者需构造恶意镜像触发，可在 QEMU 仿真环境中成功利用。
- **影响范围估算**：影响超过 50 个稳定版本及无数下游厂商分支固件，涵盖路由器、服务器、物联网设备等广泛使用 U-Boot 的嵌入式系统。
- **建议优先级**：高。建议嵌入式和物联网设备厂商立即排查 U-Boot 版本并及时更新补丁；涉及设备树解析库 fdt_get_name 函数的内存处理缺陷需重点修复。
- **相关联历史事件**：类似 Grub2 引导程序 BootHole 漏洞（CVE-2020-10713）曾影响全球 Linux 系统，引导程序层面的漏洞修复周期长、影响面广。

**3. “Muck and Load”大规模 GitHub 投毒活动曝光**
- **攻击复杂度**：中。攻击者利用 GitHub Actions 工作流自动化生成大量恶意版本，使用死信解析器技术隐藏载荷，具有一定的隐蔽性。
- **影响范围估算**：GitHub 上超 200 个仓库受影响，恶意模块累计超 700 个版本，主要针对 Go 语言开发者社区。
- **建议优先级**：高。建议 Go 语言开发者和使用第三方 Go 模块的项目立即排查 dependencies 中是否包含可疑的 dnsub-scanning-tool 及相关模块。
- **相关联历史事件**：类似事件包括 2024 年 PyPI 和 npm 包投毒事件，开源生态供应链攻击持续活跃。

**4. 《银行业保险业网络安全管理办法》公开征求意见**
- **攻击复杂度**：不适用（政策法规类）。
- **影响范围估算**：覆盖全国银行业金融机构、保险业金融机构及金融控股公司，影响金融行业网络安全合规体系建设。
- **建议优先级**：高。各金融机构应在征求意见期内（一个月）组织反馈，并对照《办法》要求评估自身网络安全治理、风险管理、外包安全及个人信息保护等合规差距。
- **相关联历史事件**：7 月 3 日已发布《金融业网络安全管理办法（征求意见稿）》，两者衔接构成金融行业网络安全监管的完整框架。

**5. 23andMe 基因数据泄露案终判赔付 4675 万美元**
- **攻击复杂度**：中。攻击利用了平台“DNA 亲属匹配”功能的连锁暴露机制。
- **影响范围估算**：约 690 万用户受影响，涉及基因数据、姓名、家谱等终身不可更改的高度敏感信息。
- **建议优先级**：中（事件已进入赔付阶段，但警示意义重大）。建议消费级基因检测服务提供商加强访问控制和数据加密，用户应警惕基因数据泄露带来的长期隐私风险。
- **相关联历史事件**：2023 年 23andMe 首次披露数据泄露事件，此次为法律终判结果。此前 MyHeritage、 ancestry.com 等同类平台也发生过数据安全事件。

# 网络安全资讯日报 – 2026-07-19

**本期要点**：

- Microsoft 发布7月安全更新，修补6个高危漏洞，其中CVE-2026-58644远程代码执行漏洞已被发现在野利用，受到香港政府CERT高威胁等级预警
- 印度最大核电站库丹库拉姆核电站发生大规模数据泄露，承包商遭勒索软件攻击致约1.9万份敏感文件外泄
- 日本物流巨头佐川急便因系统修复失误泄露约7万名会员用户信息
- 哥伦比亚国家石油公司（Ecopetrol）遭网络攻击导致数据泄露，攻击者提出勒索要求
- Google Chrome 发布紧急安全更新至版本150.0.7871.128，修复多个高危漏洞

## ⚠️ 今日重点关注

**1. Microsoft 7月安全更新：CVE-2026-58644 远程代码执行漏洞已在野利用**
- **攻击复杂度**：低，已在野积极利用，无需用户交互即可触发
- **影响范围估算**：全球范围内使用受影响 Microsoft 产品的企业及个人用户，覆盖Windows系统及多个服务器组件
- **建议优先级**：最高。香港政府CERT已发布高威胁等级警报，建议系统管理员立即排查并安装安全更新
- **相关联历史事件**：Microsoft 多次出现7月补丁日修复在野利用漏洞，如2025年7月的 CVE-2025-30514 等

**2. 印度库丹库拉姆核电站大规模数据泄露：承包商遭勒索软件攻击致1.9万份敏感文件外泄**
- **攻击复杂度**：中，通过勒索软件攻击供应链第三方承包商实现突破
- **影响范围估算**：涉及印度最大核电站的设计图纸、供应商信息和员工资料，可能影响核电站供应链安全及长期运营安全
- **建议优先级**：高。关键基础设施供应链安全风险凸显，建议各国关键基础设施运营方核查承包商安全管控
- **相关联历史事件**：2023年印度京德勒钢铁厂遭勒索攻击、2025年印度航空数据泄露等事件

**3. 哥伦比亚国家石油公司（Ecopetrol）遭网络攻击数据泄露且被勒索**
- **攻击复杂度**：中
- **影响范围估算**：哥伦比亚最大石油企业，拉美地区重要能源供应商，数据泄露范围待进一步披露
- **建议优先级**：中高。能源行业连续多日面临安全威胁，建议关注攻击者勒索手法及数据泄露规模
- **相关联历史事件**：2025年哥伦比亚能源部门遭BlackCat勒索攻击系列事件

**4. Google Chrome 紧急安全更新：修复多个高危漏洞至版本150.0.7871.128**
- **攻击复杂度**：低，远程攻击者可通过诱导用户访问恶意网页触发利用
- **影响范围估算**：全球数十亿 Chrome 浏览器用户，广泛覆盖桌面及移动端
- **建议优先级**：高。建议立即更新 Chrome 至最新版本，开启自动更新机制
- **相关联历史事件**：Google Chrome 2026年度已多次发布紧急安全更新修复在野利用漏洞

**5. 美国联邦承包商内鬼事件：被裁程序员1小时清空96个政府数据库**
- **攻击复杂度**：低（内部人员利用未及时回收的账号权限）
- **影响范围估算**：涉及45家美国联邦机构核心数据，包括国防部、国税局、国土安全部等
- **建议优先级**：中高。此事件暴露了涉密承包商在人员背景审查、账号权限回收、离职管理等方面的系统性缺陷
- **相关联历史事件**：2024年Snowflake数据泄露事件中内部凭证管理问题、SolarWinds供应链安全事件

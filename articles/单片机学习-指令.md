# STC89C52单片机指令系统总结

## 一、指令系统基本概念

STC89C52单片机所能执行的指令的集合称为指令系统。一条指令通常由两部分组成：操作码和地址码（操作数）。

STC89C52共有111条基本指令，按占用字节数分为：

- 单字节指令：49条
- 双字节指令：45条
- 三字节指令：17条

按执行时间分为：

- 1个机器周期：64条
- 2个机器周期：45条
- 4个机器周期：2条（乘、除法指令）

## 二、七种寻址方式

### 1. 立即寻址

指令中直接给出操作数，用#号表示。

```
MOV A, #60H     ; 将立即数60H送入累加器A
MOV DPTR, #3400H ; 将16位立即数3400H送入DPTR
```

### 2. 直接寻址

指令直接给出操作数地址，可访问内部RAM（00H～7FH）和特殊功能寄存器。

```
MOV A, 40H      ; 将内部RAM 40H单元内容送入A
MOV A, P0       ; 将P0口内容送入A（P0是特殊功能寄存器）
```

### 3. 寄存器寻址

操作对象为工作寄存器R0-R7、累加器A、寄存器B、DPTR和Cy位。

```
MOV A, R1       ; 将R1内容送入A
ADD A, R0       ; A与R0内容相加
```

### 4. 寄存器间接寻址

以寄存器内容为地址，以该地址中内容为操作数。使用@Ri、@DPTR等。

```
MOV A, @R0      ; 将R0指向的内部RAM单元内容送入A
MOVX A, @DPTR   ; 将DPTR指向的外部RAM单元内容送入A
```

### 5. 变址寻址

用于访问程序存储器中的数据，常用于查表操作。

```
MOVC A, @A+DPTR ; 将A+DPTR地址的内容送入A
MOVC A, @A+PC   ; 将A+PC地址的内容送入A
```

### 6. 相对寻址

用于程序转移指令，以PC当前值为基准加上偏移量形成转移地址。

```
SJMP 08H        ; 相对转移，偏移量为08H
JZ -10H         ; 累加器为0时转移，偏移量为-10H
```

### 7. 位寻址

对数据位进行操作，可寻址内部RAM 20H-2FH单元和可位寻址的特殊功能寄存器。

```
MOV C, 40H      ; 将位地址40H的内容送入进位位C
SETB P1.0       ; 将P1.0位置1
CLR 20H.5       ; 将内部RAM 20H单元的第5位清0
```

## 三、指令分类及示例

### 1. 数据传送类指令

用于数据在不同存储器间的传送，不影响标志位。

**内部数据传送：**

```
MOV A, #20H     ; 立即数送累加器
MOV R0, A       ; 累加器送寄存器
MOV 30H, 40H    ; 直接地址间数据传送
MOV @R0, #55H   ; 立即数送间接地址
```

**堆栈操作：**

```
PUSH 30H        ; 将30H单元内容压入堆栈
POP 40H         ; 将堆栈内容弹出到40H单元
```

**外部数据传送：**

```
MOVX A, @DPTR   ; 读外部RAM数据到累加器
MOVX @R0, A     ; 将累加器数据写入外部RAM
```

**查表指令：**

```
MOVC A, @A+DPTR ; 查表指令，常用于获取常数表数据
```

### 2. 算术运算类指令

包括加、减、乘、除等运算，会影响标志位。

**加法指令：**

```
ADD A, R0       ; A与R0内容相加，结果在A中
ADDC A, 30H     ; 带进位加法
INC A           ; 累加器内容加1
```

**减法指令：**

```
SUBB A, R1      ; 带借位减法
DEC 30H         ; 30H单元内容减1
```

**乘除法指令：**

```
MUL AB          ; A与B相乘，结果低8位在A，高8位在B
DIV AB          ; A除以B，商在A，余数在B
```

### 3. 逻辑运算与循环类指令

包括逻辑运算和移位操作。

**逻辑运算：**

```
ANL A, #0FH     ; A与立即数0FH进行逻辑与运算
ORL A, R0       ; A与R0进行逻辑或运算
XRL 30H, #0FFH  ; 30H单元与立即数进行逻辑异或运算
```

**移位指令：**

```
RL A            ; 累加器内容循环左移一位
RR A            ; 累加器内容循环右移一位
RLC A           ; 带进位循环左移
RRC A           ; 带进位循环右移
```

**其他逻辑操作：**

```
CLR A           ; 累加器清0
CPL A           ; 累加器内容按位取反
```

### 4. 程序转移类指令

控制程序执行顺序。

**无条件转移：**

```
LJMP 2000H      ; 长转移，可转移到64KB程序存储器任意位置
AJMP 300H       ; 绝对转移，2KB范围内转移
SJMP LABEL      ; 相对转移，相对偏移量-128~+127
```

**条件转移：**

```
JZ LABEL        ; 累加器为0时转移
JNZ LABEL       ; 累加器不为0时转移
CJNE A, #30H, LABEL ; A与立即数比较不相等时转移
DJNZ R0, LABEL  ; R0减1不为0时转移，常用于循环
```

**散转指令：**

```
JMP @A+DPTR     ; 无条件间接转移，用于程序散转
```

### 5. 子程序调用与返回指令

实现模块化程序设计。

```
LCALL SUB       ; 长调用，可调用64KB范围内子程序
ACALL SUB       ; 绝对调用，2KB范围内调用
RET             ; 子程序返回
RETI            ; 中断返回
```

### 6. 位操作指令

STC89C52具有位处理功能，可对位进行各种操作。

**位传送：**

```
MOV C, 20H.5    ; 将20H单元第5位送入进位位C
MOV 20H.3, C    ; 将进位位C送入20H单元第3位
```

**位状态修改：**

```
SETB P1.0       ; 将P1.0位置1
CLR 20H.7       ; 将20H单元第7位清0
CPL C           ; 进位位取反
```

**位逻辑运算：**

```
ANL C, 20H.5    ; 进位位与20H单元第5位进行逻辑与运算
ORL C, /20H.3   ; 进位位与20H单元第3位取反后进行逻辑或运算
```

**位条件转移：**

```
JC LABEL        ; 进位位为1时转移
JNC LABEL       ; 进位位为0时转移
JB 20H.5, LABEL ; 20H单元第5位为1时转移
JNB 20H.3, LABEL ; 20H单元第3位为0时转移
```

### 7. 空操作指令

```
NOP             ; 空操作，消耗一个机器周期，常用于延时
```

## 四、指令执行对标志位的影响

STC89C52的程序状态字寄存器PSW包含以下重要标志位：

- Cy（进位标志）
- Ac（辅助进位标志）
- OV（溢出标志）
- P（奇偶标志）

不同指令对标志位的影响不同，使用时需特别注意。

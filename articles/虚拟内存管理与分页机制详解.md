## 内存管理概述

### 什么是虚拟内存

虚拟内存是一种内存管理技术，它为每个进程提供一个独立的、连续的地址空间，使得程序可以使用比物理内存更大的地址空间。

### 虚拟内存的优势

| 优势 | 说明 |
|------|------|
| 地址空间隔离 | 每个进程有独立的地址空间 |
| 内存保护 | 防止进程间相互干扰 |
| 按需加载 | 只加载需要的页面 |
| 内存共享 | 多个进程可共享同一物理页面 |
| 灵活分配 | 不需要连续的物理内存 |

### Linux 0.11 内存管理特点

- 采用二级页表结构
- 页面大小为 4KB
- 支持写时复制（COW）
- 支持请求调页
- 最大支持 16MB 物理内存

## 分页机制基础

### 分页 vs 分段

| 特性 | 分段 | 分页 |
|------|------|------|
| 分配单位 | 可变大小 | 固定大小（4KB） |
| 碎片 | 外部碎片 | 内部碎片 |
| 地址空间 | 逻辑分段 | 线性地址空间 |
| 共享 | 段级共享 | 页级共享 |

### 地址转换过程

```
逻辑地址 (选择子:偏移)
        │
        ▼
┌───────────────────────────────┐
│      分段机制 (GDT/LDT)        │
│   将逻辑地址转换为线性地址      │
└───────────────┬───────────────┘
                │
                ▼
           线性地址 (32位)
                │
                ▼
┌───────────────────────────────┐
│       分页机制 (页表)          │
│   将线性地址转换为物理地址      │
└───────────────┬───────────────┘
                │
                ▼
           物理地址 (32位)
```

### 线性地址结构

```
线性地址 (32位):
┌─────────────────┬─────────────────┬─────────────────┐
│   页目录索引     │    页表索引      │    页内偏移      │
│    (10位)       │     (10位)      │     (12位)      │
│   31    22      │   21      12    │   11        0   │
└─────────────────┴─────────────────┴─────────────────┘

页目录索引: 选择页目录中的项 (0-1023)
页表索引:   选择页表中的项 (0-1023)
页内偏移:   页面内的字节偏移 (0-4095)
```

### 地址转换示例

```
线性地址: 0x00401025

二进制: 0000 0000 01 00 0000 0001 0000 0010 0101
        ├──────────┤ ├──────────┤ ├─────────────┤
        │页目录索引 │ │ 页表索引  │ │  页内偏移   │
        │   = 1    │ │   = 1    │ │   = 0x25   │

转换过程:
1. 从 CR3 获取页目录基址
2. 页目录项地址 = 页目录基址 + 1*4 = 页目录基址 + 4
3. 从页目录项获取页表基址
4. 页表项地址 = 页表基址 + 1*4 = 页表基址 + 4
5. 从页表项获取物理页基址
6. 物理地址 = 物理页基址 + 0x25
```

## 页表结构

### 页目录项 (PDE) 格式

```
页目录项 (4字节 = 32位):
 31              12 11  9  8  7  6  5  4  3  2  1  0
┌───────────────────┬─────┬──┬──┬──┬──┬──┬──┬──┬──┬──┐
│   页表物理地址[31:12]   │ AVL │ G│PS│ 0│ A│ PCD│PWT│U/S│R/W│P │
└───────────────────┴─────┴──┴──┴──┴──┴──┴──┴──┴──┴──┘

P (Present): 页表是否存在
R/W (Read/Write): 0=只读, 1=读写
U/S (User/Supervisor): 0=内核, 1=用户
PWT (Write-Through): 写通缓存
PCD (Cache Disable): 禁用缓存
A (Accessed): 是否被访问
PS (Page Size): 页大小 (0=4KB)
G (Global): 全局页
AVL: 系统保留位
```

### 页表项 (PTE) 格式

```
页表项 (4字节 = 32位):
 31              12 11  9  8  7  6  5  4  3  2  1  0
┌───────────────────┬─────┬──┬──┬──┬──┬──┬──┬──┬──┬──┐
│   页物理地址[31:12]    │ AVL │ G│ 0│ D│ A│ PCD│PWT│U/S│R/W│P │
└───────────────────┴─────┴──┴──┴──┴──┴──┴──┴──┴──┴──┘

P (Present): 页是否存在
R/W (Read/Write): 0=只读, 1=读写
U/S (User/Supervisor): 0=内核, 1=用户
D (Dirty): 页是否被修改
A (Accessed): 是否被访问
```

### Linux 0.11 页表项属性值

```c
/* 常用属性组合 */
#define PAGE_PRESENT    0x01    /* P=1: 页存在 */
#define PAGE_RW         0x02    /* R/W=1: 可写 */
#define PAGE_USER       0x04    /* U/S=1: 用户态可访问 */
#define PAGE_RW_USER    0x07    /* P=1, R/W=1, U/S=1 */
```

## 内存管理数据结构

### 内存映射数组

**定义位置**：**`mm/memory.c:52`**

```c
#define LOW_MEM       0x100000        /* 主内存起始地址 (1MB) */
#define PAGING_MEMORY (15*1024*1024)  /* 可分页内存大小 (15MB) */
#define PAGING_PAGES  (PAGING_MEMORY>>12)  /* 页面数量 (3840) */
#define MAP_NR(addr)  (((addr)-LOW_MEM)>>12)  /* 地址到页号映射 */

static unsigned char mem_map[PAGING_PAGES] = {0,};
```

**mem_map 数组说明**：

| 索引 | 值 | 说明 |
|------|-----|------|
| 0 | 0 | 页面空闲 |
| 1-99 | 引用计数 | 页面被引用的次数 |
| 100 | USED | 页面保留（内核使用） |

## 页面分配与释放

### get_free_page() - 分配物理页

**定义位置**：**`mm/memory.c:56-76`**

```c
unsigned long get_free_page(void)
{
register unsigned long __res asm("ax");

__asm__("std ; repne ; scasb\n\t"    /* 从后向前搜索 */
	"jne 1f\n\t"                      /* 没找到空闲页 */
	"movb $1,1(%%edi)\n\t"            /* 标记为已使用 */
	"sall $12,%%ecx\n\t"              /* 页号左移12位 */
	"addl %2,%%ecx\n\t"               /* 加上基址 LOW_MEM */
	"movl %%ecx,%%edx\n\t"
	"movl $1024,%%ecx\n\t"            /* 清零页面 */
	"leal 4092(%%edx),%%edi\n\t"
	"rep ; stosl\n\t"
	"movl %%edx,%%eax\n"              /* 返回物理地址 */
	"1:"
	:"=a" (__res)
	:"0" (0),"i" (LOW_MEM),"c" (PAGING_PAGES),
	"D" (mem_map+PAGING_PAGES-1)
	:"di","cx","dx");
return __res;
}
```

**执行流程**：

```
1. 从 mem_map 数组末尾向前搜索
2. 找到值为 0 的项（空闲页）
3. 将该项值设为 1（已使用）
4. 计算物理地址 = LOW_MEM + (页号 * 4096)
5. 清零页面内容
6. 返回物理地址
```

### free_page() - 释放物理页

**定义位置**：**`mm/memory.c:82-93`**

```c
void free_page(unsigned long addr)
{
	if (addr < LOW_MEM) return;        /* 不释放内核区域 */
	if (addr >= HIGH_MEMORY)
		panic("trying to free nonexistent page");
	
	addr -= LOW_MEM;
	addr >>= 12;                        /* 计算页号 */
	
	if (mem_map[addr]--) return;        /* 减少引用计数 */
	mem_map[addr] = 0;
	panic("trying to free free page");  /* 重复释放 */
}
```

## 页表操作

### copy_page_tables() - 复制页表

**定义位置**：**`mm/memory.c:155-191`**

```c
int copy_page_tables(unsigned long from, unsigned long to, long size)
{
	unsigned long * from_page_table;
	unsigned long * to_page_table;
	unsigned long this_page;
	unsigned long * from_dir, * to_dir;
	unsigned long nr;

	/* 检查对齐 (4MB 边界) */
	if ((from&0x3fffff) || (to&0x3fffff))
		panic("copy_page_tables called with wrong alignment");
	
	/* 计算页目录项地址 */
	from_dir = (unsigned long *) ((from>>20) & 0xffc);
	to_dir = (unsigned long *) ((to>>20) & 0xffc);
	size = ((unsigned) (size+0x3fffff)) >> 22;  /* 4MB 单位 */
	
	for( ; size-->0 ; from_dir++, to_dir++) {
		if (1 & *to_dir)
			panic("copy_page_tables: already exist");
		if (!(1 & *from_dir))
			continue;
		
		/* 获取源页表地址 */
		from_page_table = (unsigned long *) (0xfffff000 & *from_dir);
		
		/* 分配新页表 */
		if (!(to_page_table = (unsigned long *) get_free_page()))
			return -1;
		
		/* 设置页目录项 */
		*to_dir = ((unsigned long) to_page_table) | 7;
		
		/* 复制页表项数量 */
		nr = (from==0) ? 0xA0 : 1024;  /* 内核空间只复制 160 页 */
		
		for ( ; nr-- > 0 ; from_page_table++, to_page_table++) {
			this_page = *from_page_table;
			if (!(1 & this_page))
				continue;
			
			/* 清除写属性 (写时复制) */
			this_page &= ~2;
			*to_page_table = this_page;
			
			/* 增加页面引用计数 */
			if (this_page > LOW_MEM) {
				*from_page_table = this_page;
				this_page -= LOW_MEM;
				this_page >>= 12;
				mem_map[this_page]++;
			}
		}
	}
	invalidate();  /* 刷新 TLB */
	return 0;
}
```

**关键点**：

| 操作 | 说明 |
|------|------|
| 对齐检查 | 地址必须是 4MB 对齐 |
| 页目录项计算 | `(addr>>20) & 0xffc` |
| 写时复制 | 清除 R/W 位，共享物理页 |
| 引用计数 | 共享页面增加 mem_map 计数 |

### free_page_tables() - 释放页表

**定义位置**：**`mm/memory.c:95-120`**

```c
int free_page_tables(unsigned long from, unsigned long size)
{
	unsigned long * pg_table;
	unsigned long * dir, nr;

	if (from & 0x3fffff)
		panic("free_page_tables called with wrong alignment");
	if (!from)
		panic("Trying to free up swapper memory space");
	
	size = (size + 0x3fffff) >> 22;  /* 4MB 单位 */
	dir = (unsigned long *) ((from>>20) & 0xffc);
	
	for ( ; size-->0 ; dir++) {
		if (!(1 & *dir))
			continue;
		
		pg_table = (unsigned long *) (0xfffff000 & *dir);
		
		/* 释放页表中的所有页面 */
		for (nr=0 ; nr<1024 ; nr++) {
			if (1 & *pg_table)
				free_page(0xfffff000 & *pg_table);
			*pg_table = 0;
			pg_table++;
		}
		
		/* 释放页表本身 */
		free_page(0xfffff000 & *dir);
		*dir = 0;
	}
	invalidate();
	return 0;
}
```

### put_page() - 映射物理页到线性地址

**定义位置**：**`mm/memory.c:193-217`**

```c
unsigned long put_page(unsigned long page, unsigned long address)
{
	unsigned long tmp, *page_table;

	if (page < LOW_MEM || page >= HIGH_MEMORY)
		printk("Trying to put page %p at %p\n", page, address);
	if (mem_map[(page-LOW_MEM)>>12] != 1)
		printk("mem_map disagrees with %p at %p\n", page, address);
	
	/* 获取页目录项 */
	page_table = (unsigned long *) ((address>>20) & 0xffc);
	
	if ((*page_table) & 1) {
		/* 页表存在 */
		page_table = (unsigned long *) (0xfffff000 & *page_table);
	} else {
		/* 需要分配新页表 */
		if (!(tmp = get_free_page()))
			return 0;
		*page_table = tmp | 7;
		page_table = (unsigned long *) tmp;
	}
	
	/* 设置页表项 */
	page_table[(address>>12) & 0x3ff] = page | 7;
	return page;
}
```

## 页错误处理

### 页错误异常

当发生以下情况时会触发页错误异常（INT 14）：

1. 页不存在（P=0）
2. 访问权限违规（写只读页）
3. 特权级违规

### 页错误入口

**定义位置**：**`mm/page.s:12-40`**

```asm
_page_fault:
	xchgl %eax, (%esp)      ; 保存 eax，获取错误码
	pushl %ecx
	pushl %edx
	push %ds
	push %es
	push %fs
	movl $0x10, %edx        ; 内核数据段
	mov %dx, %ds
	mov %dx, %es
	mov %dx, %fs
	movl %cr2, %edx         ; 获取引起错误的线性地址
	pushl %edx              ; 参数: 线性地址
	pushl %eax              ; 参数: 错误码
	testl $1, %eax          ; 检查 P 位
	jne 1f                  ; P=1: 写保护错误
	call _do_no_page        ; P=0: 页不存在
	jmp 2f
1:
	call _do_wp_page        ; 写保护错误处理
2:
	addl $8, %esp
	pop %fs
	pop %es
	pop %ds
	popl %edx
	popl %ecx
	popl %eax
	iret
```

### 错误码格式

```
错误码格式:
┌───┬───┬───┬─────────────────┐
│ 0 │ 0 │ 0 │ P │ W │ U │     │
└───┴───┴───┴───┴───┴───┴─────┘

P (Bit 0): 0=页不存在, 1=页存在
W (Bit 1): 0=读操作, 1=写操作
U (Bit 2): 0=内核态, 1=用户态
```

### CR2 寄存器

CR2 寄存器保存引起页错误的线性地址。

## 写时复制机制

### 什么是写时复制

写时复制（Copy-on-Write, COW）是一种优化技术：
- fork() 时不立即复制物理页面
- 父子进程共享相同的物理页面
- 页面标记为只读
- 当某进程尝试写入时，才复制页面

### 写时复制流程

```
fork() 系统调用
        │
        ▼
┌─────────────────────────────────────┐
│ 复制页表，不复制物理页面             │
│ - 清除页表项的 R/W 位               │
│ - 增加物理页面的引用计数            │
└───────────────┬─────────────────────┘
                │
                ▼
        父子进程共享物理页
                │
        ┌───────┴───────┐
        ▼               ▼
   进程读取页面     进程写入页面
   (正常执行)           │
                       ▼
              ┌─────────────────┐
              │ 触发页错误异常   │
              │ (写保护错误)     │
              └────────┬────────┘
                       │
                       ▼
              ┌─────────────────┐
              │ do_wp_page()    │
              │ 分配新物理页     │
              │ 复制页面内容     │
              │ 更新页表项       │
              └────────┬────────┘
                       │
                       ▼
              进程继续写入
```

### do_wp_page() 函数

### un_wp_page() 函数

**定义位置**：**`mm/memory.c:219-237`**

```c
void un_wp_page(unsigned long * table_entry)
{
	unsigned long old_page, new_page;

	old_page = 0xfffff000 & *table_entry;
	
	/* 如果页面只有一个引用，直接设置可写 */
	if (old_page >= LOW_MEM && mem_map[MAP_NR(old_page)] == 1) {
		*table_entry |= 2;    /* 设置 R/W=1 */
		invalidate();
		return;
	}
	
	/* 分配新页面 */
	if (!(new_page = get_free_page()))
		oom();
	
	/* 减少原页面引用计数 */
	if (old_page >= LOW_MEM)
		mem_map[MAP_NR(old_page)]--;
	
	/* 更新页表项 */
	*table_entry = new_page | 7;
	invalidate();
	
	/* 复制页面内容 */
	copy_page(old_page, new_page);
}
```

## 请求调页机制

### 什么是请求调页

请求调页（Demand Paging）是一种按需加载技术：
- 程序启动时不全部加载到内存
- 当访问的页面不在内存时触发页错误
- 此时才从磁盘加载所需页面

### do_no_page() 函数

**定义位置**：**`mm/memory.c:347-379`**

```c
void do_no_page(unsigned long error_code, unsigned long address)
{
	int nr[4];
	unsigned long tmp;
	unsigned long page;
	int block, i;

	address &= 0xfffff000;  /* 页对齐 */
	tmp = address - current->start_code;
	
	/* 检查是否在代码/数据段范围内 */
	if (!current->executable || tmp >= current->end_data) {
		get_empty_page(address);  /* 分配空页 */
		return;
	}
	
	/* 尝试与其他进程共享页面 */
	if (share_page(tmp))
		return;
	
	/* 分配新页面 */
	if (!(page = get_free_page()))
		oom();
	
	/* 从可执行文件读取页面 */
	block = 1 + tmp/BLOCK_SIZE;  /* 跳过文件头 */
	for (i=0 ; i<4 ; block++, i++)
		nr[i] = bmap(current->executable, block);
	bread_page(page, current->executable->i_dev, nr);
	
	/* 清零 BSS 区域 */
	i = tmp + 4096 - current->end_data;
	tmp = page + 4096;
	while (i-- > 0) {
		tmp--;
		*(char *)tmp = 0;
	}
	
	/* 映射页面 */
	if (put_page(page, address))
		return;
	
	free_page(page);
	oom();
}
```

## 页面共享

### share_page() 函数

**定义位置**：**`mm/memory.c:318-336`**

```c
static int share_page(unsigned long address)
{
	struct task_struct ** p;

	if (!current->executable)
		return 0;
	if (current->executable->i_count < 2)
		return 0;  /* 没有其他进程共享此文件 */
	
	/* 遍历所有进程 */
	for (p = &LAST_TASK ; p > &FIRST_TASK ; --p) {
		if (!*p)
			continue;
		if (current == *p)
			continue;
		if ((*p)->executable != current->executable)
			continue;
		if (try_to_share(address, *p))
			return 1;
	}
	return 0;
}
```

### try_to_share() 函数

**定义位置**：**`mm/memory.c:300-316`**

```c
static int try_to_share(unsigned long address, struct task_struct * p)
{
	unsigned long from, to;
	unsigned long from_page, to_page;
	unsigned long phys_addr;

	/* 计算页目录项地址 */
	from_page = to_page = ((address>>20) & 0xffc);
	from_page += ((p->start_code>>20) & 0xffc);
	to_page += ((current->start_code>>20) & 0xffc);
	
	/* 获取源页表 */
	from = *(unsigned long *) from_page;
	if (!(from & 1))
		return 0;
	from &= 0xfffff000;
	from_page = from + ((address>>10) & 0xffc);
	
	/* 获取物理地址 */
	phys_addr = *(unsigned long *) from_page;
	
	/* 检查页面是否干净且存在 */
	if ((phys_addr & 0x41) != 0x01)
		return 0;
	
	phys_addr &= 0xfffff000;
	if (phys_addr >= HIGH_MEMORY || phys_addr < LOW_MEM)
		return 0;
	
	/* 获取或创建目标页表 */
	to = *(unsigned long *) to_page;
	if (!(to & 1))
		if (!(to = get_free_page()))
			oom();
		*(unsigned long *) to_page = to | 7;
	to &= 0xfffff000;
	to_page = to + ((address>>10) & 0xffc);
	
	if (1 & *(unsigned long *) to_page)
		panic("try_to_share: to_page already exists");
	
	/* 共享页面：写保护 */
	*(unsigned long *) from_page &= ~2;  /* 清除 R/W */
	*(unsigned long *) to_page = *(unsigned long *) from_page;
	invalidate();
	
	/* 增加引用计数 */
	phys_addr -= LOW_MEM;
	phys_addr >>= 12;
	mem_map[phys_addr]++;
	
	return 1;
}
```

## 内存初始化

### mem_init() 函数

**定义位置**：**`mm/memory.c:381-392`**

```c
void mem_init(long start_mem, long end_mem)
{
	int i;

	HIGH_MEMORY = end_mem;
	
	/* 将所有页面标记为已使用 */
	for (i=0 ; i<PAGING_PAGES ; i++)
		mem_map[i] = USED;
	
	/* 计算主内存起始页号 */
	i = MAP_NR(start_mem);
	
	/* 将可用内存区域标记为空闲 */
	end_mem -= start_mem;
	end_mem >>= 12;
	while (end_mem-- > 0)
		mem_map[i++] = 0;
}
```

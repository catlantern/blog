## 🔥 高危漏洞预警

> **标题**：7-Zip 曝出高危远程代码执行漏洞 CVE-2026-14266，需立即升级至 26.02 版本
> **来源**：OSCHINA、凤凰网、中关村在线
> **发布时间**：2026-07-22
> **摘要**：安全研究人员披露了影响 7-Zip 的远程代码执行漏洞 CVE-2026-14266，该漏洞存在于处理 XZ 格式压缩数据时分块结构解析存在缺陷，攻击者可构造特制压缩包诱导用户打开，触发堆缓冲区溢出并以当前用户权限执行任意代码。7-Zip 已在 26.02 版本中修复，但由于软件不支持自动更新，用户需手动前往官网升级。
> **链接**：https://www.oschina.net/news/476990
> **标签**：`#7-Zip` `#CVE-2026-14266` `#远程代码执行` `#漏洞预警`

> **标题**：Oracle 发布 2026 年 7 月关键补丁更新，CVE-2026-31431 已在野利用
> **来源**：GovCERT.HK - High Threat Security Alert (A26-07-35)
> **发布时间**：2026-07-22
> **摘要**：香港 GovCERT 发布高危安全预警，Oracle 发布 2026 年 7 月关键补丁更新（CPU），涉及 Java SE 及多个 Oracle 产品的安全漏洞。报告指出远程代码执行漏洞 CVE-2026-31431 和拒绝服务漏洞 CVE-2021-22555 已被发现在野利用，同时多个漏洞存在 PoC 利用代码，建议受影响系统立即更新。
> **链接**：https://www.govcert.gov.hk/en/alerts_detail.php?id=1975
> **标签**：`#Oracle` `#CVE-2026-31431` `#在野利用` `#CPU`

> **标题**：Mozilla Firefox 发布安全更新（MFSA2026-68/69/70），修复多个高危漏洞
> **来源**：GovCERT.HK - Security Alert (A26-07-37)
> **发布时间**：2026-07-22
> **摘要**：Mozilla 发布三份安全公告修复 Firefox 浏览器多个漏洞，影响 Firefox ESR Windows 版（需升级至 115.38 或 140.13 版本）及 Firefox Windows 版（需升级至 153 版本）。成功利用这些漏洞可能导致远程代码执行、拒绝服务、权限提升、信息泄露、安全限制绕过或欺骗攻击。
> **链接**：https://www.govcert.gov.hk/en/alerts_detail.php?id=1977
> **标签**：`#Firefox` `#Mozilla` `#浏览器安全` `#漏洞预警`

> **标题**：Google Chrome 发布安全更新至 150.0.7871.181，修复多个漏洞
> **来源**：GovCERT.HK - Security Alert (A26-07-36)
> **发布时间**：2026-07-22
> **摘要**：Google 发布安全更新修复 Chrome 浏览器多个漏洞，影响 Chrome 150.0.7871.181 之前版本。攻击者可利用精心构造的网页内容诱导用户访问，从而在受影响系统上实现远程代码执行、拒绝服务、权限提升、安全限制绕过或篡改攻击。建议用户通过自动更新或手动检查更新至最新版本。

## ⚠️ 今日重点关注

**1. 7-Zip CVE-2026-14266 远程代码执行漏洞广泛影响**
- **攻击复杂度**：低；攻击者只需构造恶意 XZ 压缩包并通过钓鱼邮件分发，用户解压即触发
- **影响范围估算**：全球数亿 7-Zip 用户，由于软件无自动更新机制，大量用户仍运行受影响版本
- **建议优先级**：极高；建议所有用户立即手动升级至 26.02 或更高版本，同时加强压缩包来源审查
- **相关联历史事件**：此前 7-Zip 曾出现过类似解压触发漏洞（如 CVE-2025-0411），本次漏洞由 Zero Day Initiative 披露

**2. 韩国外交系统数据泄露涉及约1万条外交人员信息**
- **攻击复杂度**：中等；攻击者通过 Web 应用漏洞长期渗透，从2025年4月持续至2026年2月
- **影响范围估算**：外交部总部及驻外使领馆全体前任、现任雇员，含情报与安全部门派驻人员
- **建议优先级**：高；泄露姓名、邮箱、职务等可直接用于精准钓鱼与社会工程攻击，对国家安全构成威胁
- **相关联历史事件**：2023年韩国外交部也曾遭遇网络攻击，但本次时间跨度更长、涉及人员更广，不排除国家背景黑客组织参与

**3. Oracle 7月 CPU 含在野利用漏洞 CVE-2026-31431**
- **攻击复杂度**：中高；CVE-2026-31431 为远程代码执行，已被发现在野利用
- **影响范围估算**：广泛影响 Java SE 及相关 Oracle 企业级产品部署
- **建议优先级**：极高；香港 GovCERT 发布高威胁预警，建议立即应用 Oracle 7月安全补丁
- **相关联历史事件**：Oracle 产品长期是 APT 组织重点攻击目标，此前 SolarWinds 供应链攻击中即涉及 Oracle 产品

**4. HollowGraph 新型木马利用 Microsoft 365 日历逃避检测**
- **攻击复杂度**：中等；利用合法 Microsoft Graph API 通信，传统安全工具难以识别
- **影响范围估算**：主要针对使用 Microsoft 365 的企业组织，正逐步扩散
- **建议优先级**：高；企业安全团队应加强对 Graph API 调用异常和日历事件创建的监控
- **相关联历史事件**：2024年赛门铁克曾报告黑客滥用 Graph API 嵌入木马，2025年1月发现针对 Azure AD 的高速暴力破解攻击

**5. 伊朗对亚马逊巴林数据基础设施实施导弹攻击**
- **攻击复杂度**：极高；物理导弹攻击而非网络攻击，属于国家级军事行动
- **影响范围估算**：巴林境内的亚马逊数据中心及美军通信基础设施受损
- **建议优先级**：关注；该事件表明关键云基础设施已成为地缘冲突中的实体打击目标，需重新评估数据中心的物理安全风险
- **相关联历史事件**：中东地区网络-物理混合冲突持续升级，此前胡塞武装亦对沙特实施海上封锁

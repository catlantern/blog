### 1. **8051微处理器**
- **作用**：这是单片机的核心模块，负责执行指令、处理数据、控制外围硬件的运行。
- **功能**：
  - 执行程序代码。
  - 控制单片机内部和外部的硬件资源。
  - 处理数据运算、逻辑判断等任务。

---

### 2. **存储器模块**
- **8k字节 Flash**：
  - **作用**：存储用户编写的程序代码（固化程序）。
  - **功能**：
    - 通过 **ISP**（在系统编程）可以在系统中重新写入程序。【外部程序可以在无需拆卸单片机芯片的情况下更新芯片中的程序】
    - 通过 **IAP**（应用编程）可以在运行过程中动态修改存储器内容。【内部程序可以自由的更新芯片】

- **512字节 SRAM**：
  - **作用**：存储运行期间的临时数据。
  - **功能**：
    - 用作程序的工作内存，保存变量、堆栈等。

- **E²PROM（Data Flash）**：
  - **作用**：存储需要长期保存的数据（如配置信息、运行状态）。
  - **功能**：
    - 支持动态读写，数据断电后仍然保存。

---

### 3. **输入输出接口**
- **四个 8位并行端口（P0、P1、P2、P3）**：
  - **作用**：提供数字输入输出功能，与外部硬件设备交互。
  - **功能**：
    - 控制外部设备（如 LED、继电器）。
    - 接收外部信号（如按键、传感器）。

- **P4口（附加的 I/O 端口）**：
  - **作用**：扩展额外的输入输出功能。
  - **功能**：
    - 提供额外的 I/O 引脚，支持更多设备连接。

---

### 4. **定时器和串口**
- **3个定时器**：
  - **作用**：提供时间管理功能。
  - **功能**：
    - 定时触发事件。
    - 实现时间间隔控制（如延时、周期性任务）。
    - 用于计数外部事件（如脉冲信号）。

- **UART（串口）**：
  - **作用**：实现串行通信。
  - **功能**：
    - 与外部设备通信（如电脑、传感器、其他单片机）。
    - 发送和接收数据。

---

### 5. **双数据指针**
- **作用**：用于快速操作内存数据。
- **功能**：
  - 提供两个独立的数据指针，方便程序在不同内存地址间快速切换。
  - 提高数据处理效率。

---

### 6. **看门狗**
- **作用**：监控程序运行状态，防止程序跑飞或死机。
- **功能**：
  - 如果程序长时间没有响应，看门狗会触发系统复位。
  - 提高系统的稳定性和可靠性。

```python
import matplotlib.pyplot as plt
from matplotlib.patches import Rectangle
from matplotlib.animation import FuncAnimation
from matplotlib.widgets import Button

# 创建画布
fig, ax = plt.subplots(figsize=(12, 8))
ax.set_xlim(0, 14)
ax.set_ylim(0, 10)
ax.axis('off')
plt.title("单片机架构数据传输流程动画", fontsize=16)

# 存储组件和连接线
components = []
connections = []
animation_running = False  # 动画状态开关

# ---------------------- 绘制组件 ----------------------
def create_component(x, y, width, height, color, label, fontsize=10):
    """创建一个组件（矩形）并添加到图中"""
    rect = Rectangle((x, y), width, height, facecolor=color, edgecolor='black', linewidth=2)
    ax.add_patch(rect)
    text = ax.text(x + width / 2, y + height / 2, label, ha='center', va='center', fontsize=fontsize)
    components.append((rect, text, (x + width / 2, y + height / 2)))  # 存储中心坐标
    return (rect, text, (x + width / 2, y + height / 2))

# 绘制各个组件
acc, _, acc_center = create_component(4, 6, 1.5, 1, '#c8e6c9', 'ACC\n累加器', 10)
alu, _, alu_center = create_component(6, 6, 2, 1.5, '#a8d1ff', 'ALU\n算术逻辑单元', 12)
tmp1, _, tmp1_center = create_component(9, 7, 1.5, 1, '#c8e6c9', 'TMP1', 10)
tmp2, _, tmp2_center = create_component(9, 5, 1.5, 1, '#c8e6c9', 'TMP2', 10)
ram, _, ram_center = create_component(2, 4, 2, 1.5, '#ffd9b3', 'RAM', 12)
flash, _, flash_center = create_component(11, 6, 2, 1.5, '#ffccd5', '程序存储器\n(Flash)', 10)
control, _, control_center = create_component(6, 8.5, 2, 1.2, '#d1d1e0', '控制单元', 12)
port, _, port_center = create_component(9, 3, 2, 1.5, '#fff9c4', 'I/O端口', 10)
pc, _, pc_center = create_component(11, 8.5, 2, 1, '#e8f5e9', '程序计数器\n(PC)', 10)
ir, _, ir_center = create_component(11, 4.5, 2, 1, '#e8f5e9', '指令寄存器\n(IR)', 10)

# ---------------------- 绘制数据线 ----------------------
def draw_connection(start, end, color='gray', linestyle='-', linewidth=1.5):
    """绘制两个组件之间的连接线"""
    line, = ax.plot([start[0], end[0]], [start[1], end[1]], color=color, linestyle=linestyle, linewidth=linewidth)
    connections.append(line)
    return line

# 绘制所有数据线
conn_acc_alu = draw_connection(acc_center, (alu_center[0] - 0.5, alu_center[1]))
conn_alu_tmp1 = draw_connection((alu_center[0] + 0.5, alu_center[1]), tmp1_center)
conn_alu_tmp2 = draw_connection((alu_center[0] + 0.5, alu_center[1]), tmp2_center)
conn_ram_acc = draw_connection(ram_center, acc_center)
conn_tmp1_port = draw_connection(tmp1_center, port_center)
conn_flash_control = draw_connection(flash_center, control_center)
conn_control_alu = draw_connection(control_center, alu_center)
conn_pc_flash = draw_connection(pc_center, flash_center)
conn_flash_ir = draw_connection(flash_center, ir_center)
conn_ir_control = draw_connection(ir_center, (control_center[0], control_center[1] - 0.3))

# 存储所有连接以便高亮
connection_list = [
    conn_acc_alu, conn_alu_tmp1, conn_alu_tmp2, conn_ram_acc,
    conn_tmp1_port, conn_flash_control, conn_control_alu,
    conn_pc_flash, conn_flash_ir, conn_ir_control
]

# ---------------------- 动画逻辑 ----------------------
# 数据传输步骤：(连接索引, 描述)
transmission_steps = [
    (3, "RAM → ACC: 读取数据"),
    (0, "ACC → ALU: 传输操作数"),
    (6, "控制单元 → ALU: 传输控制信号"),
    (1, "ALU → TMP1: 传输运算结果"),
    (4, "TMP1 → I/O端口: 输出结果"),
    (7, "程序计数器 → Flash: 地址信号"),
    (8, "Flash → 指令寄存器: 传输指令"),
    (9, "指令寄存器 → 控制单元: 传输指令"),
    (5, "Flash → 控制单元: 传输数据")
]

# 添加描述文本
desc_text = ax.text(7, 0.3, "", ha='center', va='center', fontsize=12, 
                    bbox=dict(facecolor='white', edgecolor='black', pad=5))

# 重置所有连接的样式
def reset_connections():
    for conn in connection_list:
        conn.set_color('gray')
        conn.set_linewidth(1.5)
        conn.set_linestyle('-')

# 更新函数
def update(frame):
    reset_connections()
    conn_index, description = transmission_steps[frame % len(transmission_steps)]
    current_conn = connection_list[conn_index]
    current_conn.set_color('#ff5722')
    current_conn.set_linewidth(3)
    desc_text.set_text(description)
    return connection_list + [desc_text]

# ---------------------- 按钮控制 ----------------------
def start_animation(event):
    global animation_running
    if not animation_running:
        ani.event_source.start()
        animation_running = True

# 创建按钮
ax_button = plt.axes([0.8, 0.01, 0.1, 0.05])  # 按钮位置
btn_start = Button(ax_button, '开始动画')
btn_start.on_clicked(start_animation)

# 创建动画
ani = FuncAnimation(
    fig, update,
    frames=len(transmission_steps),
    interval=2000,
    blit=True,
    repeat=True
)
ani.event_source.stop()  # 初始时暂停动画

plt.tight_layout()
plt.show()

```
